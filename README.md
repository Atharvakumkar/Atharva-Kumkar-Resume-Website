## Overview

This is a fully client-side resume website hosted on Vercel. It serves as a digital alternative to a traditional PDF resume, offering a clean and accessible way to showcase professional information.

## Getting Started

### Prerequisites

- A modern web browser
- (Optional) Node.js if you want to run a local dev server

### Run Locally

Clone the repository:

```bash
git clone https://github.com/Atharvakumkar/Atharva-Kumkar-Resume-Website.git
cd Atharva-Kumkar-Resume-Website/resume
```

Open `index.html` directly in your browser, or serve it with a simple local server:

```bash
# Using Python
python -m http.server 8000
```

Then visit `http://localhost:8000` in your browser.

## Deployment

The site is deployed via [Vercel](https://vercel.com) and is publicly accessible at:

```
https://atharva-kumkar-resume.vercel.app
```

To deploy your own fork:

1. Fork this repository
2. Sign in to [Vercel](https://vercel.com)
3. Import your forked repository
4. Set the root directory to `resume/` (if prompted)
5. Deploy — Vercel handles the rest automatically
